Language Visualization

This project helps analyze text quickly by organizing sentences into topic groups automatically. Paste in any English (only, for now) text, select the number clusters, then click "Send Text to Server."

I love NLP and I built this in a weekend. I wanted to share it because I just found myself using it to analyze a text that looked too long/intimidating that I wanted to understand faster than I could read it.
